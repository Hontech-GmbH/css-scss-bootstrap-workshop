01 - Responsive Media Query Layout

Create a responsive layout for the screen breakpoints 768px, 1024px, 1280px
Create a mixin that calculates a flex-item width based on the amount of flex items and the gap size. (Alternatively just use size - gap-size, e.g. 20% - 20px)
Make the Flex container hold the following amount of elements on a single line

Size # of elements
< 768px 1 Element
< 1024px 2 Elements
< 1280px 3 Elements
>= 1280px 4 Element

Make the Grid have 1 column when size < 768, 2 when size < 1024 and 3 when size >= 1024
